# Core Brand Assets

This directory contains the original hero and background assets for StrainSpotter.

## Required Files

1. **hero.png**
   - Original StrainSpotter hero emblem
   - Used in landing pages and key sections
   - Recommended size: 512×512 or higher

2. **strainspotter-bg.jpg**
   - Main background image for the application
   - Used in body backgrounds globally
   - Recommended size: 1920×1080 or higher
   - Should be optimized for web (compressed JPG)

## Usage

These files should be copied from the original assets or restored from backup.

## Fallback

If files are missing, the app will use CSS gradients as fallback:
```css
background: linear-gradient(135deg, #0A1A12 0%, #111416 100%);
```

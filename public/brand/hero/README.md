# Botanical OS Hero Banners

This directory should contain high-resolution hero banner images for Botanical OS.

## Required Files

1. **botanical-hero-banner.png** (2560×1440)
   - Main hero banner for landing pages
   - Dark forest gradient (#0A1A12 → #111416)
   - Subtle terpene vapor lines
   - Emerald bioluminescent particle glows
   - Surreal macro "bud texture constellation" motifs
   - Soft depth of field blur layers

2. **botanical-hero-1.png** (2560×1440)
   - Alternative hero for Pro Tools dashboard
   - Similar style with different composition

3. **botanical-hero-2.png** (2560×1440)
   - Alternative hero for Garden intro
   - More "friendly" surface tones

## Usage

- Landing page: `/brand/hero/botanical-hero-banner.png`
- Pro Tools dashboard: `/brand/hero/botanical-hero-1.png`
